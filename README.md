# ATLAS Books

## Installation

1. Download project from GitHub
    ```
    git clone https://github.com/Simetraa/library.git
    ```
2. Install dependencies
    ```
    npm install
    ```
   ```
   composer install
   ```
3. Seed test data
    ```
    php artisan migrate:fresh --seed
    ```
4. Run the server
    ```
    php artisan serve
    ```
